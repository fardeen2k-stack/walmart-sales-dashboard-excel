
# Walmart Sales Dashboard (Excel Project)

This project analyzes Walmart sales data from 2011–2014 using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Metrics

## Dashboard Features
- Total Sales KPI
- Total Quantity KPI
- Total Profit KPI
- Sales by State
- Profit by State
- Sales by City
- Yearly Sales Trend
- Interactive slicers for filtering

## Key Insights
- California generated the highest sales and profit
- Some states showed negative profit
- Sales increased steadily from 2011 to 2014
